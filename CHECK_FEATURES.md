## 🐛 已知潜在问题

1. **编辑功能可能的问题**
   - 编辑正在运行的任务可能导致状态不一致
   - 建议：只允许编辑未运行的任务

2. **搜索框焦点**
   - Cmd+F 快捷键未实现
   - 需要添加：聚焦到搜索框

3. **导入验证**
   - 没有验证导入的 JSON 格式
   - 可能导入损坏的数据

## 💡 快速修复建议

如果发现任何问题，可以：

1. **清空浏览器缓存**
   ```
   localStorage.clear()
   ```

2. **重启应用**
   ```bash
   cd /Users/huaaan/huaan-command
   # 关闭应用
   NODE_ENV=development npm run tauri dev
   ```

3. **检查控制台**
   - 按 Cmd+Option+I 打开开发者工具
   - 查看 Console 是否有错误
