## 📝 配置说明

### 首次使用
1. 打开 DMG 文件
2. 将 "Huaan Command" 拖入 Applications 文件夹
3. 首次打开可能需要在系统设置中允许运行
4. 配置 AI 设置（设置 -> AI 设置）
   - 输入 API Key
   - 设置 API Base URL（可选）
   - 选择模型

### AI 配置
- **OpenAI**:
  - API Key: `sk-...`
  - Base URL: 默认或留空
  - 模型: `gpt-4o-mini` 等

- **DeepSeek**:
  - API Key: `sk-...`
  - Base URL: `https://api.deepseek.com`
  - 模型: `deepseek-chat` 或 `deepseek-coder`

## 📋 使用建议

### 终端使用
1. 创建新标签页开始工作
2. 使用 Warp 模式切换终端/AI 模式
3. 在 AI 模式下直接对话
4. 使用 @ 选择文件/目录

- 明确描述需求
- 提供足够的上下文
- 使用连续对话保持语境

### 任务管理
- 创建任务跟踪项目
- 使用任务状态管理进度
- 查看任务历史了解执行情况
