# Huaan Command - 发布说明

## 最新版本: v1.2.0

**发布日期**: 2025-11-16

### �� 核心功能

#### 智能终端
- ✅ 多标签页终端支持
- ✅ 会话自动保存和恢复
- ✅ 终端编号智能管理
- ✅ 工作目录自动追踪
- ✅ 命令历史记录
- ✅ 反向搜索（Ctrl+R）
- ✅ zsh 作为默认 Shell

#### CLI 工具集成
- ✅ 快速启动常用 CLI 工具
  - Claude Code Assistant
  - Google Gemini
  - iFlow CLI
  - GitHub Copilot
  - 等等...

#### 主题系统
- ✅ 6 种内置主题
  - 系统跟随
  - 暗色/亮色
  - Ocean / Forest / Sunset
- ✅ 自定义颜色支持
- ✅ 实时主题切换

#### 终端设置
- ✅ 字体大小调整
- ✅ 字体系列选择
- ✅ 光标闪烁控制
- ✅ 会话自动恢复

### 🚀 性能特性

- **启动速度**: < 2 秒
- **内存占用**: ~80MB (空闲)
- **响应延迟**: < 100ms
- **终端数量**: 无限制

### 📦 安装包信息

#### macOS
- **文件**: `Huaan Command_1.2.0_x64.dmg`
- **架构**: Intel x86_64
- **系统要求**: macOS 10.15+

### 🔧 技术栈

- **前端**: Vue 3 + Vite + Pinia
- **终端**: xterm.js 5.5.0
- **后端**: Tauri 2.9.0 + Rust
- **PTY**: portable-pty
