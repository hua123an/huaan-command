```vue

├─ Status 标签页

## 🎨 设计特点

### 配色体系
```
Primary    #00b4ff  蓝色 (主要操作、活跃状态)
Success    #4caf50  绿色 (暂存、新增)
Error      #ff6b6b  红色 (错误、删除)
Warning    #ff9800  橙色 (未暂存、警告)
Neutral    #999     灰色 (次要信息)
Background #1e1e1e  深灰 (VS Code 风格)
```

### UI 组件
```
按钮       + 正方形/圆形边框，带 hover 效果
输入框     + 深色背景，蓝色焦点状态
标签页     + 活跃时底部蓝色条形
进度条     + 渐变色 (蓝->绿)
卡片       + 微妙的阴影和渐变背景
```

```

git branch -a                   列出分支
git show --stat                 显示提交详情
